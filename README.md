# Núcleo de Projetos Socioambientais

Portal institucional desenvolvido para divulgação dos projetos socioambientais da Universidade La Salle Canoas.

## Sobre o Projeto

O Portal de Projetos Socioambientais tem como objetivo apresentar iniciativas de pesquisa, extensão e inovação ambiental desenvolvidas pela Universidade La Salle Canoas e instituições parceiras.

O portal reúne informações sobre projetos estratégicos, áreas de atuação, parceiros, resultados e impactos socioambientais.

---

## Funcionalidades

- Página institucional responsiva

- Navegação fixa com efeito de scroll

- Integração com mapas

---

## Projetos Apresentados

### RestaurAPA

Projeto desenvolvido na Área de Proteção Ambiental do Ibirapuitã, voltado à recuperação de áreas degradadas por espécies invasoras.

### Nosso Itapuã

Projeto de restauração ecológica no Parque Estadual de Itapuã com foco no controle de espécies exóticas invasoras e recuperação de ecossistemas naturais.

### CRIEC Canoas – SbN

Projeto voltado à implementação de Soluções Baseadas na Natureza (SbN) para fortalecimento da resiliência climática no município de Canoas.

## Como Atualizar Conteúdos

### Alterar textos

Os textos dos projetos e seções institucionais podem ser alterados diretamente no arquivo:

```text
index.html
```

### Alterar imagens

Substitua os arquivos dentro da pasta:

```text
assets/images/
```

mantendo os mesmos nomes ou atualizando os caminhos no HTML.

### Alterar mapas

Os mapas incorporados utilizam Google Maps Embed e podem ser modificados substituindo o link do iframe correspondente.

---

## Publicação

O projeto é publicado automaticamente através do Vercel.

Após realizar alterações:

```bash
git add .
git commit -m "Descrição da alteração"
git push
```

O Vercel executará uma nova publicação automaticamente.
